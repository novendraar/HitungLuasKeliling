# Hitung Luas dan Keliling Bangunan

Selamat datang di aplikasi pertama saya menggunakan Python! Aplikasi ini dirancang untuk membantu Anda menghitung luas dan keliling berbagai bentuk bangunan 2D dengan mudah.

## Fitur Utama

Aplikasi ini mendukung perhitungan untuk berbagai bentuk bangunan 2D, termasuk:

1. **Persegi** - Hitung luas dan keliling dari sebuah persegi berdasarkan panjang sisi.
2. **Persegi Panjang** - Hitung luas dan keliling dari sebuah persegi panjang berdasarkan panjang dan lebar.
3. **Segitiga Sama Sisi** - Hitung luas dan keliling dari sebuah segitiga sama sisi dengan memasukkan sisi, alas, dan tinggi.
4. **Lingkaran** - Hitung luas dan keliling dari sebuah lingkaran berdasarkan jari-jari.

## Cara Menggunakan

1. **Pilih Bangunan**: Pilih jenis bangunan yang ingin Anda hitung dari menu dropdown.
2. **Pilih Jenis Hitungan**: Pilih apakah Anda ingin menghitung luas atau keliling.
3. **Masukkan Parameter**: Masukkan nilai yang diperlukan sesuai dengan jenis bangunan dan jenis hitungan yang dipilih.
4. **Klik Hitung**: Tekan tombol 'Hitung' untuk mendapatkan hasil perhitungan.
